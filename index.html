Sarc7

Evaluating Sarcasm Detection and Generation with Seven Types and Emotion-Informed Techniques

About the Paper

Sarc7 introduces a benchmark with seven fine-grained sarcasm types and emotion-informed modeling. It enables nuanced evaluation for both classification and generation. Emotion and linguistic cues help models better interpret sarcastic intent.

Paper Overview

Abstract

We introduce Sarc7, a sarcasm benchmark dataset with subtype classification and generation tasks. The dataset contains emotion-informed labels, fine-grained sarcasm types, and enables multi-task training for interpretability.

Related Work

Past sarcasm datasets focus on binary detection. Sarc7 goes beyond this by including pragmatic and emotional factors, along with subtype-level distinctions that reflect natural sarcasm use.

Benchmark Features

  • 7 sarcasm subtypes (e.g. hyperbole, rhetorical question)
  • Emotion annotation & multi-task labels
  • Supports generation & classification

Methodology

We annotated social media posts using sarcasm typologies and emotion cues. Transformer models were trained on both subtype classification and sarcasm generation using emotional embeddings.

Results

Emotion-aware models outperformed all baselines. Generation tasks improved significantly with emotion guidance, and subtype modeling improved interpretability and generalization.
