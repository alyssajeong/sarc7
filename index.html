Evaluating Sarcasm Detection and Generation with Seven Types and Emotion-Informed Techniques

About the Paper

Sarc7 introduces a sarcasm benchmark with seven distinct sarcasm types, emotion-informed modeling, and a dataset annotated with nuanced pragmatic cues. The work combines classification and generation tasks and evaluates models across sarcasm subtasks to push sarcasm understanding beyond binary detection.

Paper Overview

Abstract

We propose Sarc7, a new benchmark for evaluating sarcasm detection and generation with fine-grained type labels and emotional cues. The dataset enables multi-task learning and targeted modeling of sarcasm styles across rhetorical, emotional, and semantic boundaries.

Related Works

Prior sarcasm datasets have focused on binary labeling. Sarc7 builds on these by incorporating pragmatic theory, emotion modeling, and fine-grained annotations to better reflect sarcasmâ€™s complexity in natural discourse.

Benchmark Features

  • 7 sarcasm subtypes: hyperbole, understatement, rhetorical question, etc.
  • Emotion-informed annotation & model tasks
  • Multi-task learning setup (detection + generation)

Methodology

We annotated posts using a taxonomy informed by linguistic theory and emotion psychology. Models are trained for subtype classification and sarcasm generation using transformer-based architectures with emotion embeddings.

Results

Emotion-informed models consistently outperform baselines. Sarcasm subtype classification improves overall interpretability, and generation outputs better reflect nuanced sarcasm intent.
